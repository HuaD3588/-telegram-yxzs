#Telegram 谢幕营销助手｜多账号管理｜用户采集｜批量发送｜强拉入群｜养号工具
###账号管理
- 多账号导入/管理：支持协议号导入、直登号（tdata）导入、手动登录
- 
###用户采集
- 普通采集：从群成员列表采集（适用于成员可见的群/超级群）
- 隐藏群采集：最近 N 条 / 最近 N 天范围 最少发言次数阈值（过滤低质量路人） 随机轻量延迟（更稳定）
- 导出：自动导出采集结果（txt）+ 任务报告（json）
- 
- ###批量群发
- 多账号批量发送内容（按设置节奏/延迟执行）
- 失败记录与任务报告导出
- 
###批量强拉
- 多账号把指定用户名/用户列表拉入目标群
- 任务可控、结果统计与报告导出
- 
###养号系统
- 自动浏览/已读、对话数/消息数/间隔可配置
- 支持时间窗口、黑名单关键词过滤
- 任务状态面板可视化
